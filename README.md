# Premium Tic Tac Toe

A beautiful, single-file Tic Tac Toe game built using modern HTML, CSS (Vanilla), and JavaScript. 

Designed with modern glassmorphism dark aesthetics, glowing ambient shadows, responsive layout scaling, and fluid pop-in animations.

## Features

- **Interactive 3x3 Board**: Clickable responsive layout adapting gracefully to mobile and desktop displays.
- **Ambient Glows & Neon Icons**: Vibrant neon-cyan indicators for Player **X** and neon-pink for Player **O**.
- **Robust Gameplay Checking**: Instantly detects and highlights winning row, column, or diagonal combinations, as well as tie games (draws).
- **Restart functionality**: Clear board states cleanly and reset turns with the click of a button.
- **Zero Dependencies**: Pure, standard ES6 JavaScript and CSS, requiring no external packages or builders.

## How to Play

1. Clone this repository or download the [index.html](index.html) file.
2. Double-click [index.html](index.html) to open it directly in any web browser.
3. Player X goes first, followed by Player O. 
4. Aim for 3-in-a-row to win!
